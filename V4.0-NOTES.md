# v4.0 Release Notes

## Major Updates

1. Created new homepage at the root path featuring:
   - Overview of all three products (VoiceAgent, VoiceLens, PredictML)
   - Company info sections matching the Executive Summary slide
   - Consistent design theme with the rest of the site

2. Restructured routes:
   - Root path now has a landing page for all products
   - Configured redirect from root to /VoiceAgent
   - Moved VoiceAgent content to /VoiceAgent path

3. Added Hospital Appointments animation to PredictML page:
   - Animated banner showing appointment show/no-show prediction
   - Replaced the rocket animation
   - Maintained consistent styling with the site

4. Fixed various import/export issues

## How to Use

To use this version, checkout tag v4.0:
```
git checkout v4.0
npm install
npx next dev -p 3003
```

Access the site at http://localhost:3003
